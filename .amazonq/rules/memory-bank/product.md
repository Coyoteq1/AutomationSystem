# CrowbaneArena Mod - Product Overview

## Purpose
CrowbaneArena is a V Rising server mod that creates enhanced arena gameplay experiences with progression management and V Blood tracking systems. It provides a controlled environment for PvP combat while maintaining player progression integrity.

## Key Features

### Arena System
- **Zone-based Arena Management**: Configurable arena zones with entry/exit points and spawn locations
- **God-Mode Integration**: Full spellbook and hotbar access within arena boundaries
- **Territory Management**: Arena grid system for spatial control and player tracking
- **Auto-Entry Service**: Automatic arena entry detection and management

### Progression Management
- **Snapshot System**: Captures and restores player progression states
- **V Blood Tracking**: Monitors and manages V Blood progression for arena participants
- **Equipment Management**: Handles loadouts, weapons, armor sets, and builds
- **Inventory Control**: Manages player inventories during arena sessions

### Configuration System
- **JSON-based Configuration**: Flexible arena settings via `arena_config.json`
- **Loadout Management**: Customizable weapon and armor configurations
- **Dynamic Reloading**: Hot-reload capability for configuration changes
- **Prefab Remapping**: Custom item and equipment mapping system

### Integration Features
- **BepInEx Plugin Architecture**: Native V Rising server integration
- **Harmony Patching**: Runtime game behavior modification
- **VampireCommandFramework**: Command system integration for admin controls
- **Unity ECS Compatibility**: Works with V Rising's Entity Component System

## Target Users
- **V Rising Server Administrators**: Setting up competitive arena environments
- **PvP Communities**: Organized combat events and tournaments
- **Mod Developers**: Extending arena functionality with additional features
- **Server Operators**: Managing player progression in controlled environments

## Value Proposition
Provides a comprehensive arena management solution that maintains game balance while enabling competitive gameplay, with robust progression tracking and flexible configuration options for diverse server needs.
